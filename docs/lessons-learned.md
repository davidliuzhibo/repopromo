# Lessons Learned

## 1. Do not start with one-click video

Static review assets must come first.

## 2. Raw SVG text is fragile for dense Chinese pages

Use HTML/CSS instead.

## 3. The hardest part is story compression

The video usually fails before rendering because the content is not structured well.

## 4. Main slides and supporting shots must play different roles

- Main slides = narrative skeleton
- Supporting shots = emphasis

## 5. Never overwrite review assets

Always version output files and archive older ones into old/.
